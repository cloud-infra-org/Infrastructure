# Security Policy

We take the security of this infrastructure repository seriously.
This document explains how to report security concerns related to infrastructure.

## Scope
This repository contains configurations that manage environments and platforms.
Security issues related to access, permissions, or configuration are in scope.

## Reporting a Security Issue
Do not report infrastructure security issues in public issues or discussions.
Report them privately to the infrastructure maintainers.
Include details about the affected environment and potential impact.

## What to Include
- A clear description of the security concern
- The environment or area affected
- Steps to reproduce, if applicable

## Response Process
The team will review and validate the report.
High-risk issues will be fixed as a priority.
You will be informed once the issue is addressed.

Thank you for helping keep our infrastructure secure.
